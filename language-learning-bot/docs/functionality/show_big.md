### Конфигурация изображений

```yaml
word_images:
  enabled: true
  fonts:
    word_size: 80           # Размер шрифта для слова
    transcription_size: 40  # Размер шрифта для транскрипции
  colors:
    background: [255, 255, 255]  # Белый фон
    text: [50, 50, 50]           # Темно-серый текст
```
